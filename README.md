# Botanical Blue - Premium Wedding Invitation Framework

Botanical Blue is a luxury white-label wedding invitation framework built with Next.js. It features a cinematic opening experience, a motion design system, and an emotional audio player.

## Features
- **Zero Hardcoding**: All data is driven by a single `invitation.json` file.
- **Cinematic Opening**: Premium envelope reveal with natural light and wax seal.
- **Motion System**: Consistent scroll-based reveal animations.
- **Responsive & Accessible**: Pixel-perfect from 320px to 1920px. Keyboard accessible.
- **White-label Ready**: Built for studios and freelancers to resell to clients.

## Quick Start
1. Clone the repository.
2. Run `npm install`.
3. Start the server with `npm run dev`.
4. Edit `data/invitation.json` to customize content.
